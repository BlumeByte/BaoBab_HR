# Baobab HR (Flutter Web)

Local-only BambooHR-inspired HR web application.

## Features
- Login with any non-empty email and password (local only)
- Sidebar navigation for all primary HR modules
- White + blue theme with dark mode toggle in Settings
- Editable profile name and profile image URL in Settings
- No Firebase auth, no Create Account screen, and no subscription plan flows

## Run
```bash
flutter pub get
flutter run -d chrome
```
